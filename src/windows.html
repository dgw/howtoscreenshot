Print Screen

This is the most commonly known method of taking a screenshot. The Print Screen key can be combined with other keys to change its behaviour.

With this method, you do not need to paste the image data into a program like Microsoft Paint, as it creates a file directly on your computer.

1

Press the Windows and Print Screen keys on your keyboard.

Note: Some laptops need the fn key held in order to use the Print Screen key.

2

Your screenshot can be found in the C:\Users\<yourname>\Pictures\Screenshots folder.

Note: You can copy/paste %USERPROFILE%\Pictures\Screenshots into the Explorer location bar to go straight to this folder.

Snipping Tool

The Snipping Tool was added to Windows during Windows Vista, and has remained in the Operating System since then. It's main feature is being able to take screenshots of parts of the screen, instead of capturing the entire area.

1

Open the Start menu and search for Snipping Tool.

2

Select the New button in the menu bar.

3

Click and drag to select the area to capture a screenshot.

Note: See Extra: Mode for more ways of selecting the area to capture.

4

Select the Save button in the menu bar to save the captured image to your computer.

Extra: Annotating

Before saving, you can draw on the image to create annotations or hide personal information.

Select the Pen, Highlighter, and Eraser buttons in the menu bar to use the tools.

Extra: Mode

Selecting the Mode buttons in the menu bar allows you to change the capture mode.

The default mode is "Rectangular Snip". You can also take a screenshot of a specific window ("Window Snip"), the entire screen ("Fullscreen Snip"), or a drawn shape ("Free-form Snip").
